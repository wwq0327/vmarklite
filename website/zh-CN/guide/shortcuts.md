# 键盘快捷键

VMark 专为键盘优先的工作流而设计。大多数快捷键可在设置中自定义。少数原语是固定的：多光标选择器 `Mod+D`（选择下一个出现位置）和 `Mod+Shift+L`（选择所有出现位置），以及全局的撤销/重做绑定。其他多光标快捷键（跳过出现位置、软撤销光标、向上/向下添加光标）均可配置。标记为 _(上下文感知)_ 的快捷键由编辑器内部针对特定结构处理（例如任务列表复选框切换），不在自定义注册表中暴露。

## 符号说明

- **Mod** = macOS 上的 Cmd，Windows/Linux 上的 Ctrl
- **Alt** = macOS 上的 Option

## macOS 上的功能键

VMark 使用功能键（F4–F10）进行快速模式切换。在 macOS 上，这些键默认映射到系统功能（亮度、音量等）。

**不按 Fn 直接使用 F 键：**

1. 打开 **系统设置** → **键盘**
2. 启用 **"将 F1、F2 等键用作标准功能键"**

或者，按 **Fn** 键的同时按 F4–F10 来触发 VMark 快捷键。

::: tip
如果你希望保留系统功能键，可以在设置中（`Mod + ,`）将 VMark 快捷键自定义为其他按键组合。
:::

## 智能 Tab 导航

Tab 和 Shift+Tab 具有上下文感知能力 —— 可以跳出括号、引号、格式标记和链接。

| 上下文 | Tab 操作 |
|--------|---------|
| 在 `)`、`]`、`}`、引号之前 | 跳过闭合字符 |
| 在中日韩括号 `」`、`』` 等之前 | 跳过闭合括号 |
| 在 **粗体**、*斜体*、`代码`内部 | 跳至格式标记之后 |
| 在链接内部 | 跳至链接之后 |

| 上下文 | Shift+Tab 操作 |
|--------|---------------|
| 在 `(`、`[`、`{`、引号之后 | 跳到开括号之前 |
| 在中日韩括号 `「`、`『` 等之后 | 跳到开括号之前 |
| 在 **粗体**、*斜体*、`代码`内部 | 跳到格式标记之前 |
| 在链接内部 | 跳到链接之前 |

::: tip
完整指南（包括中日韩括号、弯引号和设置）请参阅[智能 Tab 导航](/zh-CN/guide/tab-navigation)。
:::

## 自定义快捷键

1. 使用 `Mod + ,` 打开设置
2. 导航到 **快捷键** 选项卡
3. 点击任意快捷键进行编辑
4. 按下你想要的按键组合
5. 更改自动保存

::: tip
快捷键在适用情况下会与菜单加速键同步，因此菜单项会显示你自定义的快捷键。
:::
