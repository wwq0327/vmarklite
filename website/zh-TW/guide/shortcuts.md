# 鍵盤快捷鍵

VMark 以鍵盤優先的工作流程設計。大多數快捷鍵均可在設定中自訂。少數基本快捷鍵為固定值：多重游標選擇器 `Mod+D`（選取下一個符合項目）與 `Mod+Shift+L`（選取所有符合項目），以及全域的還原/重做綁定。其他多重游標快捷鍵（跳過符合項目、軟性還原游標、在上方/下方新增游標）均可自訂。標示為 _(情境感知)_ 的快捷鍵由編輯器內部針對特定結構處理（例如切換任務核取方塊），不會出現在自訂註冊表中。

## 符號說明

- **Mod** = macOS 上的 Cmd，Windows/Linux 上的 Ctrl
- **Alt** = macOS 上的 Option

## macOS 上的功能鍵

VMark 使用功能鍵（F4–F10）進行快速模式切換。在 macOS 上，這些鍵預設對應至系統功能（亮度、音量等）。

**不按住 Fn 鍵直接使用 F 鍵：**

1. 開啟 **系統設定** → **鍵盤**
2. 啟用 **「將 F1、F2 等鍵用作標準功能鍵」**

或者，按住 **Fn** 鍵再按 F4–F10，以觸發 VMark 快捷鍵。

::: tip
如果你偏好將系統功能保留在 F 鍵上，可以在設定（`Mod + ,`）中自訂 VMark 快捷鍵，使用不同的按鍵組合。
:::

## 智慧 Tab 導覽

Tab 和 Shift+Tab 具有情境感知功能 — 可在括號、引號、格式標記與連結之間快速跳脫。

| 情境 | Tab 操作 |
|------|----------|
| 位於 `)`、`]`、`}`、引號之前 | 跳過右括號字元 |
| 位於中日韓文括號 `」`、`』` 等之前 | 跳過右括號 |
| 位於 **粗體**、*斜體*、`程式碼` 之內 | 跳至格式之後 |
| 位於連結之內 | 跳至連結之後 |

| 情境 | Shift+Tab 操作 |
|------|----------------|
| 位於 `(`、`[`、`{`、引號之後 | 跳至左括號之前 |
| 位於中日韓文括號 `「`、`『` 等之後 | 跳至左括號之前 |
| 位於 **粗體**、*斜體*、`程式碼` 之內 | 跳至格式之前 |
| 位於連結之內 | 跳至連結之前 |

::: tip
完整指南請參閱[智慧 Tab 導覽](/zh-TW/guide/tab-navigation)，包含中日韓文括號、彎引號與設定說明。
:::

| 操作 | 快捷鍵 |

::: tip
`Mod + C` 在沒有選取範圍時，會向正在執行的程序傳送 SIGINT。完整指南請參閱[整合終端機](/zh-TW/guide/terminal)。
:::

## 自訂快捷鍵

1. 使用 `Mod + ,` 開啟設定
2. 導覽至 **快捷鍵** 分頁
3. 點選任意快捷鍵以編輯
4. 按下你想要的按鍵組合
5. 變更會自動儲存

::: tip
快捷鍵在適用時會與選單快捷鍵同步，因此選單項目會顯示你自訂的快捷鍵。
:::
