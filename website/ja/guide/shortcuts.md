# キーボードショートカット

VMark はキーボードファーストのワークフロー向けに設計されています。ほとんどのショートカットは設定でカスタマイズできます。一部のプリミティブは固定されています：マルチカーソルセレクター `Mod+D`（次の出現を選択）と `Mod+Shift+L`（すべての出現を選択）、およびグローバルな元に戻す／やり直しのバインディング。その他のマルチカーソルショートカット（出現をスキップ、カーソルをソフトアンドゥ、上／下にカーソルを追加）はカスタマイズ可能です。_(コンテキスト依存)_ と表示されているショートカットは、特定の構造（例: タスクリストのチェックボックス切り替え）のためにエディタ内で処理され、カスタマイズレジストリには公開されていません。

## 表記

- **Mod** = macOS では Cmd、Windows/Linux では Ctrl
- **Alt** = macOS では Option

## macOS のファンクションキー

VMark はクイックモード切り替えにファンクションキー（F4〜F10）を使用します。macOS では、これらのキーはデフォルトでシステム機能（輝度、音量など）にマッピングされています。

**Fn キーを押さずに F キーを直接使用するには:**

1. **システム設定** → **キーボード** を開く
2. **「F1、F2 などのキーを標準的なファンクションキーとして使用する」** を有効にする

または、F4〜F10 を押す際に **Fn** キーを押しながら使用して VMark のショートカットをトリガーします。

::: tip
システム機能を F キーに維持したい場合、設定（`Mod + ,`）で VMark のショートカットを別のキーの組み合わせにカスタマイズできます。
:::

## スマートタブナビゲーション

Tab と Shift+Tab はコンテキストに応じて動作します — 括弧、引用符、フォーマットマーク、リンクからエスケープできます。

| コンテキスト | Tab アクション |
|------------|-------------|
| `)`、`]`、`}`、引用符の前 | 閉じる文字を飛び越える |
| CJK 括弧 `」`、`』` などの前 | 閉じる括弧を飛び越える |
| **太字**、*斜体*、`コード` の中 | フォーマットの後にジャンプ |
| リンクの中 | リンクの後にジャンプ |

| コンテキスト | Shift+Tab アクション |
|------------|------------------|
| `(`、`[`、`{`、引用符の後 | 開く文字の前にジャンプ |
| CJK 括弧 `「`、`『` などの後 | 開く括弧の前にジャンプ |
| **太字**、*斜体*、`コード` の中 | フォーマットの前にジャンプ |
| リンクの中 | リンクの前にジャンプ |

::: tip
CJK 括弧、カーリー引用符、設定を含む完全なガイドは[スマートタブナビゲーション](/ja/guide/tab-navigation)を参照してください。
:::

セレクションなしの `Mod + C` は実行中のプロセスに SIGINT を送信します。完全なガイドは[統合ターミナル](/ja/guide/terminal)を参照してください。
:::

## ショートカットのカスタマイズ

1. `Mod + ,` で設定を開く
2. **ショートカット** タブに移動する
3. 任意のショートカットをクリックして編集する
4. 希望のキーの組み合わせを押す
5. 変更は自動的に保存される

::: tip
ショートカットは該当する場合はメニューアクセラレーターと同期するため、メニューアイテムにカスタマイズされたショートカットが表示されます。
:::
