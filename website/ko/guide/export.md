# 내보내기 및 인쇄

VMark는 문서를 내보내고 공유하는 여러 방법을 제공합니다.

## 내보내기 모드

### 폴더 모드 (기본값)

깔끔한 구조의 자체 포함 폴더를 만듭니다:

```text
MyDocument/
├── index.html
└── assets/
    ├── image1.png
    ├── image2.jpg
    └── ...
```

**장점:**
- 서비스 시 깔끔한 URL (`/MyDocument.html` 대신 `/MyDocument/`)
- 단일 폴더로 쉽게 공유
- 단순한 에셋 경로 (`assets/image.png`)
- 정적 사이트 호스트와 잘 작동

### 단일 파일 모드

단일 자체 포함 HTML 파일을 만듭니다:

```text
MyDocument.html
```

모든 이미지가 데이터 URI로 포함되어 완전히 이식 가능하지만 파일 크기가 더 큽니다.

## 내보내는 방법

### HTML 내보내기

1. **파일 → HTML 내보내기** 사용
2. 내보내기 위치 선택
3. 폴더 모드: 폴더 이름 입력 (예: `MyDocument`)
4. 단일 모드: `.html` 확장자와 함께 파일명 입력

### 인쇄 / PDF 내보내기

1. `Cmd/Ctrl + P`를 누르거나 **파일 → 인쇄** 사용
2. 시스템 인쇄 대화상자에서 인쇄하거나 PDF로 저장

### 다른 형식으로 내보내기

VMark는 [Pandoc](https://pandoc.org/) — 범용 문서 변환기 — 과 통합하여 마크다운을 추가 형식으로 내보낼 수 있습니다. 메뉴에서 직접 형식을 선택합니다:

**파일 → 내보내기 → 다른 형식 →**

| 메뉴 항목 | 확장자 |
|----------|--------|
| Word (.docx) | `.docx` |
| EPUB (.epub) | `.epub` |
| LaTeX (.tex) | `.tex` |
| OpenDocument (.odt) | `.odt` |
| Rich Text (.rtf) | `.rtf` |
| 일반 텍스트 (.txt) | `.txt` |

**설정:**

1. [pandoc.org/installing](https://pandoc.org/installing.html)에서 Pandoc을 설치하거나 패키지 관리자를 통해 설치합니다:
   - macOS: `brew install pandoc`
   - Windows: `winget install pandoc`
   - Linux: `apt install pandoc`
2. VMark를 재시작하거나 **설정 → 파일 및 이미지 → 문서 도구** 로 이동하여 **감지** 클릭
3. **파일 → 내보내기 → 다른 형식 → [형식]** 을 사용하여 내보내기

Pandoc이 설치되어 있지 않으면 다른 형식 하위 메뉴 하단에 **"Pandoc 필요 — pandoc.org"** 링크가 표시됩니다.

**설정 → 파일 및 이미지 → 문서 도구** 에서 Pandoc이 감지되었는지 확인할 수 있습니다.

### HTML로 복사

`Cmd/Ctrl + Shift + C`를 눌러 렌더링된 HTML을 클립보드에 복사하여 다른 응용 프로그램에 붙여넣습니다.

## VMark Reader

HTML로 내보낼 때 (스타일 모드), 문서에는 **VMark Reader** — 강력한 기능을 갖춘 대화형 독서 경험 — 가 포함됩니다.

### 설정 패널

기어 아이콘 (오른쪽 하단) 클릭 또는 `Esc`를 눌러 설정 패널을 엽니다:

| 설정 | 설명 |
|------|------|
| 글꼴 크기 | 텍스트 크기 조정 (12px – 24px) |
| 줄 높이 | 줄 간격 조정 (1.2 – 2.0) |
| 테마 | 테마 전환 (White, Paper, Mint, Sepia, Night) |
| CJK-라틴 간격 | CJK와 라틴 문자 사이 간격 토글 |

### 목차

TOC 사이드바는 긴 문서 탐색을 돕습니다:

- **토글**: 패널 헤더 클릭 또는 `T` 누르기
- **탐색**: 제목을 클릭하여 이동
- **키보드**: `↑`/`↓` 화살표로 이동, `Enter`로 이동
- **하이라이트**: 스크롤 시 현재 섹션이 강조 표시됨

### 읽기 진행 상황

페이지 상단의 미묘한 진행 막대가 문서를 얼마나 읽었는지 보여줍니다.

### 맨 위로

스크롤을 내리면 플로팅 버튼이 나타납니다. 클릭하거나 `Home`을 눌러 맨 위로 돌아갑니다.

### 이미지 라이트박스

이미지를 클릭하면 전체 화면 라이트박스에서 볼 수 있습니다:

- **닫기**: 바깥쪽 클릭, `Esc` 누르기, 또는 X 버튼 클릭
- **탐색**: 여러 이미지의 경우 `←`/`→` 화살표 사용
- **줌**: 이미지가 원래 크기로 표시됨

### 코드 블록

각 코드 블록에는 대화형 컨트롤이 포함됩니다:

| 버튼 | 기능 |
|------|------|
| 줄 번호 토글 | 이 블록의 줄 번호 표시/숨기기 |
| 복사 버튼 | 코드를 클립보드에 복사 |

복사 버튼은 성공 시 체크 표시를 표시합니다.

### 각주 탐색

각주는 완전히 대화형입니다:

- 각주 참조 `[1]`을 클릭하면 정의로 이동
- `↩` 역참조를 클릭하면 읽던 위치로 돌아감

### 키보드 단축키

| 키 | 동작 |
|----|------|
| `Esc` | 설정 패널 토글 |
| `T` | 목차 토글 |
| `↑` / `↓` | TOC 항목 탐색 |
| `Enter` | 선택한 TOC 항목으로 이동 |
| `←` / `→` | 라이트박스에서 이미지 탐색 |
| `Home` | 맨 위로 스크롤 |

## 내보내기 단축키

| 동작 | 단축키 |
|------|--------|
| HTML 내보내기 | _(메뉴만)_ |

## 팁

### 내보낸 HTML 서비스하기

폴더 내보내기 구조는 모든 정적 파일 서버와 잘 작동합니다:

```bash
# Python
cd MyDocument && python -m http.server 8000

# Node.js (npx)
npx serve MyDocument

# 직접 열기
open MyDocument/index.html
```

### 오프라인 보기

두 가지 내보내기 모드 모두 완전히 오프라인으로 작동합니다:

- **폴더 모드**: 모든 브라우저에서 `index.html` 열기
- **단일 모드**: `.html` 파일 직접 열기

수학 방정식 (KaTeX)은 스타일시트를 위해 인터넷 연결이 필요하지만, 다른 모든 내용은 오프라인으로 작동합니다.

### 모범 사례

1. **공유하거나 호스팅할 문서에는 폴더 모드** 사용
2. **이메일이나 채팅을 통한 빠른 공유에는 단일 모드** 사용
3. **접근성을 위해 설명적인 이미지 대체 텍스트** 포함
4. **다양한 브라우저에서 내보낸 HTML 테스트**
