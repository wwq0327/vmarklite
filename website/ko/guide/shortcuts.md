# 키보드 단축키

VMark는 키보드 우선 워크플로우를 위해 설계되었습니다. 대부분의 단축키는 설정에서 사용자 정의할 수 있습니다. 일부 기본 단축키는 고정되어 있습니다: 멀티 커서 선택자 `Mod+D` (다음 항목 선택)와 `Mod+Shift+L` (모든 항목 선택), 그리고 전역 실행 취소/다시 실행 바인딩. 다른 멀티 커서 단축키 (항목 건너뛰기, 소프트 커서 실행 취소, 위/아래에 커서 추가)는 구성 가능합니다. _(컨텍스트 인식)_ 으로 표시된 단축키는 특정 구조 (예: 작업 목록 체크박스 토글)에 대해 에디터 내에서 처리되며 사용자 정의 레지스트리에 노출되지 않습니다.

## 표기법

- **Mod** = macOS에서 Cmd, Windows/Linux에서 Ctrl
- **Alt** = macOS에서 Option

## macOS의 기능 키

VMark는 빠른 모드 전환을 위해 기능 키 (F4–F10)를 사용합니다. macOS에서 이러한 키는 기본적으로 시스템 기능 (밝기, 볼륨 등)에 매핑됩니다.

**Fn 키를 누르지 않고 F 키를 직접 사용하려면:**

1. **시스템 설정** → **키보드** 를 엽니다
2. **"F1, F2 등 키를 표준 기능 키로 사용"** 활성화

또는 F4–F10를 눌러 VMark 단축키를 실행할 때 **Fn** 키를 누르고 있습니다.

::: tip
시스템 기능을 F 키에 유지하는 것을 선호한다면, 설정 (`Mod + ,`)에서 VMark 단축키를 다른 키 조합으로 사용자 정의할 수 있습니다.
:::

### F 키 빠른 참조

| 키 | 동작 |
|----|------|
| `F2` | 다음 문제 |
| `Shift + F2` | 이전 문제 |
| `F4` | 줄 오름차순 정렬 |
| `Shift + F4` | 줄 내림차순 정렬 |
| `F5` | 소스 미리보기 |
| `F6` | 소스 모드 토글 |
| `F7` | 상태 표시줄 토글 |
| `F8` | 포커스 모드 |
| `F9` | 타이프라이터 모드 |
| `F10` | 읽기 전용 모드 |

## 스마트 탭 탐색

Tab과 Shift+Tab은 상황 인식적입니다 — 괄호, 따옴표, 서식 마크, 링크를 탈출합니다.

| 컨텍스트 | Tab 동작 |
|----------|----------|
| `)`, `]`, `}`, 따옴표 앞 | 닫는 문자 건너뛰기 |
| CJK 괄호 `」`, `』` 등 앞 | 닫는 괄호 건너뛰기 |
| **굵게**, *기울임꼴*, `코드` 안 | 서식 후로 이동 |
| 링크 안 | 링크 후로 이동 |

| 컨텍스트 | Shift+Tab 동작 |
|----------|----------------|
| `(`, `[`, `{`, 따옴표 뒤 | 여는 문자 앞으로 이동 |
| CJK 괄호 `「`, `『` 등 뒤 | 여는 괄호 앞으로 이동 |
| **굵게**, *기울임꼴*, `코드` 안 | 서식 전으로 이동 |
| 링크 안 | 링크 전으로 이동 |

::: tip
CJK 괄호, 중괄호 따옴표, 설정을 포함한 전체 가이드는 [스마트 탭 탐색](/ko/guide/tab-navigation)을 참조하세요.
:::

## 터미널

통합 터미널이 포커스되어 있을 때:

| 동작 | 단축키 |
|------|--------|
| 터미널 토글 | `` Ctrl + ` `` |
| 복사 | `Mod + C` (선택 시) |
| 붙여넣기 | `Mod + V` |
| 지우기 | `Mod + K` |
| 검색 | `Mod + F` |

터미널 검색 바가 열려 있을 때:

| 동작 | 단축키 |
|------|--------|
| 다음 일치 | `Enter` |
| 이전 일치 | `Shift + Enter` |
| 검색 닫기 | `Escape` |

::: tip
선택 없이 `Mod + C`를 사용하면 실행 중인 프로세스에 SIGINT를 보냅니다. 전체 가이드는 [통합 터미널](/ko/guide/terminal)을 참조하세요.
:::

## 단축키 사용자 정의

1. `Mod + ,`로 설정을 엽니다
2. **단축키** 탭으로 이동합니다
3. 편집할 단축키를 클릭합니다
4. 원하는 키 조합을 누릅니다
5. 변경사항이 자동으로 저장됩니다

::: tip
단축키는 해당하는 경우 메뉴 액셀러레이터와 동기화되므로 메뉴 항목에 사용자 정의된 단축키가 표시됩니다.
:::
